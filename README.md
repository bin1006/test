


# RaiDrive 사용법

**1. RaiDrive 다운로드 방법 알아보기**
<br />
- [다운로드](https://www.raidrive.com/ko/download)를 클릭하여 파일을 다운로드한 후 실행합니다.

> 다운로드한 후 실행하면 아래와 같은 화면이 나옵니다.<br />
> 그림에 표시된 **추가** 버튼을 **클릭**하세요.

<br />
![main](/main.jpg?raw=true)
<br />
